While standard text analysis stops at basic sentiment, this project reconstructs the "forma mentis" (cognitive architecture) of LLMs. By combining an 8-emotion psychometric framework with Textual Forma Mentis Networks (e.g., semantic centrality, topological density), it quantifies how generative models internally structure concepts, biases, and affective associations from unstructured output alone.

It applies wherever synthetic text requires deep cognitive profiling—such as benchmarking LLM mental models, auditing behavioral personas, or evaluating cognitive alignment in human-AI interactions.

Because suspiciously high accuracy often flags data leakage or synthetic artifacts, the pipeline integrates systematic feature ablation and SHAP validation to rigorously interpret which network topological signals actually drive the LLM's cognitive classification.

**What this shows:** turning raw text (such as user reviews for a product, LLM synthetic text) into quantitative interpretable signal, and validating it before trusting it.
